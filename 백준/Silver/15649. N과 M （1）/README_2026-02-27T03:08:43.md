# [Silver III] N과 M (1) - 15649 

[문제 링크](https://www.acmicpc.net/problem/15649) 

### 분류

백트래킹

### 문제 설명

<p>자연수 N과 M이 주어졌을 때, 아래 조건을 만족하는 길이가 M인 수열을 모두 구하는 프로그램을 작성하시오.</p>

<ul>
	<li>1부터 N까지 자연수 중에서 중복 없이 M개를 고른 수열</li>
</ul>

### 입력 

 <p>첫째 줄에 자연수 N과 M이 주어진다. (1 ≤ M ≤ N ≤ 8)</p>

### 출력 

 <p>한 줄에 하나씩 문제의 조건을 만족하는 수열을 출력한다. 중복되는 수열을 여러 번 출력하면 안되며, 각 수열은 공백으로 구분해서 출력해야 한다.</p>

<p>수열은 사전 순으로 증가하는 순서로 출력해야 한다.</p>

# 💻 코드 리뷰
처음 문제를 읽고 어떤식으로 접근할지에대해서 고민해본 결과  
단순 반복문으로 접근하기에는 어려움이 존재하다고 판단해 dfs 알고리즘을 통해 문제를 해결하기로 하였다.  
코드의 핵심은 백트래킹이라는 개념인데 아래 링크를 통해 자세한 개념은 읽어보기를 바란다. 

먼저 초기코드에서는 1 ~ n까지의 숫자들중에서 갈이가 m인 수열을 모두 구하기 위해서   
dfs함수를 선언하여주었고 배열도 m만큼의 크기로 선언해주었다.  
dfs알고리즘은 재귀 함수로 자기자신을 호출하여 반복하는 알고리즘이다.  
재귀함수의 종료조건을 depth==m으로 설정하여 m크기만큼의 수열이 만들어지면 출력하는 방식으로 작성하였다.

이후 종료 조건이 만족하면 출력하고 함수를 반환하는 식으로 작동한다.  
하지만 초기코드에는 문제가 존재하였는데 바로 출력한적있는 숫자를 또 출력하다는 것이다.  
예를 들어서 문제에서는 1 2 / 1 3 / 1 4 이런식의 출력을 요구하지만
초기 코드에서는 1 1/ 1 2/ 1 3/ 1 4 와 같은 식으로 출력한다.  
왜 그런지에 대해 고민해본 결과 함수를 다시 호출하게되면 어떤 조건없이는 중복이 될 수 밖에 없는 구조임을 확이하였다.  
그래서 visited라는 방문처리 배열을 전역변수로 선언하여 1~n까지의 숫자중 수열에 포함된 숫자를 방문처리하여 문제를 해결하였다.

여기서 visited[i]=true부터 visited[i]=false 해당 부분이 백트래킹을 하는 과정이다.
아래 링크를 통해서 자세히 설명하겠지만 간단하게 말하자면 지금 과정을 선택하기 전으로 돌아간다. 즉 1 2를 출력하고 방문처리가 되어 있으면 
1과 2는 더이상 출력할수없다. 하지만 m 크기의 모든 수열을 출력하려면 1 2 뿐만아니라 2 1 / 3 1 / 1 3 이런식으로 여러번 방문을 해주어야한다.
그래서 1 2를 출력하고 visted[]=true 되어있던것을 false로 다시 돌려주는 과정을 거치는 것이고 이를 백트래킹이라고 부른다.

 ## 🏆 메모 

 오랜만에 dfs를 사용한 문제를 풀어 떠올리는데 시간이 걸린문제였다. 특히 전체 구조와 흐름은 쉽게 작성했지만  
 방문처리에대한 내용과 같은 세부적인 내용을 떠올리기 쉽지 않았다.   
 하지만 dfs알고리즘을 연습하기에 좋은 문제이고 dfs의 개념을 정확하게 알고 사용할 수 있다면   
 쉽게 접근 가능할 것이라고 생각된다.
