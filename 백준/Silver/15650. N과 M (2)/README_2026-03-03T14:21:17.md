### 문제 설명

<p>자연수 N과 M이 주어졌을 때, 아래 조건을 만족하는 길이가 M인 수열을 모두 구하는 프로그램을 작성하시오.</p>

<ul>
	<li>1부터 N까지 자연수 중에서 중복 없이 M개를 고른 수열</li>
	<li>고른 수열은 오름차순이어야 한다.</li>
</ul>

### 입력 

 <p>첫째 줄에 자연수 N과 M이 주어진다. (1 ≤ M ≤ N ≤ 8)</p>

### 출력 

 <p>한 줄에 하나씩 문제의 조건을 만족하는 수열을 출력한다. 중복되는 수열을 여러 번 출력하면 안되며, 각 수열은 공백으로 구분해서 출력해야 한다.</p>

<p>수열은 사전 순으로 증가하는 순서로 출력해야 한다.</p>

#  🚀  추가 코드
```diff
import java.util.*;

public class Main{
    static int arr[];
 
    static StringBuilder sb=new StringBuilder();
    public static void main(String args[]){
        Scanner s=new Scanner(System.in);
        
        int n=s.nextInt();
        int m=s.nextInt();
        arr=new int[m];

        dfs(n,m,0,0);
        System.out.print(sb);
    }
    public static void dfs(int n,int m,int depth,int num){
        if(depth==m){
            for(int i=0;i<m;i++){
                sb.append(arr[i]).append(" ");
            }sb.append("\n");
            return;
        }

        // 중복이 불가능하게하면 다쓴 수는 방문처리를 해주어야함
        // 현재 코드는 백트래킹하면서 방문처리를 다시 돌림
        for(int i=num;i<n;i++){
                arr[depth]=i+1;
                dfs(n,m,depth+1,i+1);
            }
    }
}
```
# 💻 코드 리뷰

해당 코드는 visited로 별도의 방문처리없이 문제를 해결하는 방식으로  
dfs 호출시 num값에 +1을 해주는 것으로 중복을 방지해주었다.


 ## 🏆 메모 
처음 문제를 읽고 중복처리말고는 이전에 풀었던 N과M(1) 문제와 전체적인 구조나 사용하는 알고리즘이 같아서 어렵지 않게 접근하였다.  
하지만 중복처리하는 과정에서 처음에는 n=4 m=2 라고 하였을때 1을 사용다하면 방문처리 유지한다 이런식으로 접근하려했는데  
백트래킹을 해주는 과정에서 해당 방식은 불가능하다고 판단해 dfs를 호출할때 현재 값을 넘겨주어 처음에는 0부터 시작하는데 
0을 다돌고 나면 1로 오게되어 자연스럽게 앞에서 사용했던 0은 방문처리가 되는 방식으로 코드를 작성하였다.
