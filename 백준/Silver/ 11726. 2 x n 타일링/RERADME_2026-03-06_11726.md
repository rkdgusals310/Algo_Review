### 문제 설명

<p>2×n 크기의 직사각형을 1×2, 2×1 타일로 채우는 방법의 수를 구하는 프로그램을 작성하시오.</p>

<p>아래 그림은 2×5 크기의 직사각형을 채운 한 가지 방법의 예이다.</p>

<p style="text-align: center;"><img alt="" src="https://onlinejudgeimages.s3-ap-northeast-1.amazonaws.com/problem/11726/1.png" style="height:50px; width:125px"></p>

### 입력 

 <p>첫째 줄에 n이 주어진다. (1 ≤ n ≤ 1,000)</p>

### 출력 

 <p>첫째 줄에 2×n 크기의 직사각형을 채우는 방법의 수를 10,007로 나눈 나머지를 출력한다.</p>



#  🚀  오답코드

```diff
import java.util.*;

public class Main{
    public static void main(String args[]){
        Scanner s=new Scanner(System.in);
        
        int n=s.nextInt();
        
        System.out.print(dp(n)%10007);
    }
    public static int dp(int num){
        if(num==1){return 1;}
        else if(num==2){return 2;}
        
        return dp(num-1)+dp(num-2);
        
    }
}
```

#  🚀  수정코드
```diff
import java.util.*;

public class Main{
    public static int dpN[];
    public static void main(String args[]){
        Scanner s=new Scanner(System.in);
        
        int n=s.nextInt();
        dpN=new int[n+1];
        Arrays.fill(dpN,-1);
        
        System.out.print(dp(n)%10007);
    }
    public static int dp(int num){
        if(dpN[num]!=-1){return dpN[num];}
        else if(num==2){return 2;}
        else if(num==1){return 1;}
        return dpN[num]=(dp(num-1)+dp(num-2))%10007;
        
    }
}
```

 ## 🏆 메모 

오랜만에 DP 문제를 풀이해보아서 시간이 조금 걸렸던 문제이다.  
예전에 DP알고리즘을 공부한적이 있어 재귀함수를 활용하는 Top-Down 방식과 반복문을 사용하는 Bottom-Up방식이 존재한다는 것을 기억하고 있었다.  
본인은  Bottom-Up방식보다 Top-Down방식이 더 어렵게 느껴져 연습을 위해 Top-Down방식으로 문제를 해결했었다.  
그래서 이번에도 Top-Down방식으로 문제를 접근하였고 재귀함수를 사용해 코드를 작성하였다.    
하지만 문제를 제출한 결과는 계속 시간초과였고 이유를 모르겠어서 DP에 대한 개념을 다시 확인해보았다.   
확인해본 결과 본인은 재귀함수라는 부분에만 매몰되어 DP가 아닌 재귀함수를 작성해 시간초과가 나온것이었다.    
여기서 본인이 빼놓은 부분이 DP에서 가장 중요한 부분이었고 그 부분은 바로 "메모제이션 "이었다.  
중복호출을 방지해 불필요한 낭비를 막아주는 부분이었는데 이부분을 빼먹고 구현하려다보니 시간초과가 발생한것이었다.
