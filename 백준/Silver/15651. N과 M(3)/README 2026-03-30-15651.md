### 문제 설명

<p>자연수 N과 M이 주어졌을 때, 아래 조건을 만족하는 길이가 M인 수열을 모두 구하는 프로그램을 작성하시오.</p>

<ul>
	<li>1부터 N까지 자연수 중에서 M개를 고른 수열</li>
	<li>같은 수를 여러 번 골라도 된다.</li>
</ul>

### 입력 

 <p>첫째 줄에 자연수 N과 M이 주어진다. (1 ≤ M ≤ N ≤ 7)</p>

### 출력 

 <p>한 줄에 하나씩 문제의 조건을 만족하는 수열을 출력한다. 중복되는 수열을 여러 번 출력하면 안되며, 각 수열은 공백으로 구분해서 출력해야 한다.</p>

<p>수열은 사전 순으로 증가하는 순서로 출력해야 한다.</p>

#  🚀 최종 코드 

```java
// 순열 문제인데 중복가능하다. 
import java.util.*;

public class Main{
    public static int arr[];
    public static boolean visited[]; // 삭제 
    public static StringBuilder sb=new StringBuilder();
    
    public static void main(String args[]){
        Scanner s=new Scanner(System.in);
        
        int n=s.nextInt();
        int m=s.nextInt();
        
        arr=new int[m];
        visited=new boolean[n]; // 삭제  불필요한 부분임;
        
        dfs(n,m,0);
        System.out.print(sb);
    }
    public static void dfs(int n,int m,int depth){
        if(depth==m){
            for(int i=0;i<depth;i++){
                sb.append(arr[i]).append(" ");
            }sb.append("\n");
            return;
        }
        for(int i=0;i<n;i++){
            arr[depth]=i+1;
            dfs(n,m,depth+1);
        }
        
    }
}
```
문제를 읽고 dfs를 활용한 중복순열 문제라고 생각했다.    
dfs를 활용한 문제들은 기본적인 틀이 존재하는데, 방문처리를 위한 배열을 활용한다거나 백트래킹을 활용한다거나 여러가지 틀이 존재한다.  
위의 코드에서도 그런 기본적인 틀을 작성하고 문제를 풀이했다. 전체적인 틀은 dfs함수를 선언하는데 depth를 통해 m개를 선택하는 경우의 수를 구하는 것이다.  
하지만 해당 문제는 중복을 허용하기 때문에 방문처리가 필요없다.     
처음에 기본적인 틀을 작성한다고 아무생각없이 작성했지만 이부분은 사용할 필요가 없다는 것을 dfs 함수를 작성하면서 깨달았다.  


## 🏆 메모
dfs 알고리즘의 기본 문제들중 가장 좋아하는 시리즈 문제이다. 3이외의 다른 번호의 N과 M 문제들도 풀이해보았는데,
