### 문제 설명

<p>괄호 문자열(Parenthesis String, PS)은 두 개의 괄호 기호인 ‘(’ 와 ‘)’ 만으로 구성되어 있는 문자열이다. 그 중에서 괄호의 모양이 바르게 구성된 문자열을 올바른 괄호 문자열(Valid PS, VPS)이라고 부른다. 한 쌍의 괄호 기호로 된 “( )” 문자열은 기본 VPS 이라고 부른다. 만일 x 가 VPS 라면 이것을 하나의 괄호에 넣은 새로운 문자열 “(x)”도 VPS 가 된다. 그리고 두 VPS x 와 y를 접합(concatenation)시킨 새로운 문자열 xy도 VPS 가 된다. 예를 들어 “(())()”와 “((()))” 는 VPS 이지만 “(()(”, “(())()))” , 그리고 “(()” 는 모두 VPS 가 아닌 문자열이다. </p>

<p>여러분은 입력으로 주어진 괄호 문자열이 VPS 인지 아닌지를 판단해서 그 결과를 YES 와 NO 로 나타내어야 한다. </p>

### 입력 

 <p>입력 데이터는 표준 입력을 사용한다. 입력은 T개의 테스트 데이터로 주어진다. 입력의 첫 번째 줄에는 입력 데이터의 수를 나타내는 정수 T가 주어진다. 각 테스트 데이터의 첫째 줄에는 괄호 문자열이 한 줄에 주어진다. 하나의 괄호 문자열의 길이는 2 이상 50 이하이다. </p>

### 출력 

 <p>출력은 표준 출력을 사용한다. 만일 입력 괄호 문자열이 올바른 괄호 문자열(VPS)이면 “YES”, 아니면 “NO”를 한 줄에 하나씩 차례대로 출력해야 한다. </p>


#  🚀  오답 코드 

```diff
/*
    괄호 ( )로만 구성되어 있는 문자열 
    올바른 괄호  문자열  VPS라고 함
    한쌍의 괄호 기호로 된 () 문자열은 기본 VPS
    
*/
//(())())
import java.util.*;

public class Main{
    public static void main(String args[]){
        Stack<Character> stack=new Stack<>();
        Scanner s=new Scanner(System.in);
        StringBuilder sb=new StringBuilder();
        String answer="YES";
        
        int n=s.nextInt();
        boolean breakN=false;
        for(int i=0;i<n;i++){
            String str=s.next();
            for(int j=0;j<str.length();j++){
                char c=str.charAt(j);
                if(stack.isEmpty()&&c=='('){
                    stack.push(c);
                }
                else{
                    if(stack.peek()=='('&&c==')')stack.pop();
                    else if(stack.peek()=='('&&c=='(')stack.push(c);
                    else {
                        breakN=true;
                        break;
                    }
                }
            }
            if(breakN)answer="NO";
            else answer="YES";
            sb.append(answer).append("\n");
        }
        System.out.print(sb);
    }
}
```
해당 코드는 최소의 작성한 코드로 Stack를 사용해야한다는 부분을 파악하고 문자열을 처음부터 차례대로 비교해 조건에 맞게 stack에 push하거나 pop하는 방식으로 작성하였다.  
조건은 아래와 같이 크게 2가지로 나누어진다.

1. stack이 비어있는 경우
2. stack이 비어있지 않은 경우

비어있는 경우에는 비교하는 문자가 '('인 경우에만 push해주고 아닌 경우는 NO로 판단한다.    
이유는 () 쌍으로 구성되어 있는 문자열을 충족하려면 시작은 무조건 '('인 경우만 쌍 판단이 가능하기 때문이다.    

비어있지 않은 경우에는 3가지 경우를 비교해야한다.    
먼저  stack의 마자믹 요소가 '('인지를 비교한다. '('인경우 비교 문자가 '('면 push , ')'이면 쌍이 맞는 경우이므로 pop해준다.  
stack의 마지막 요소가 ')'인 경우는 존재하지 않는다. 왜냐하면 stack에 push를 하는경우는 문자가 '('인 경우만이고      
')'인 경우에는 쌍이 만족했다고 판단해 pop해주기 때문에 stack의 마지막 요소는 ')'이 될 수가 없다.    
하지만 반복문을 미리 종료 시킬 수 있기때문에 breakN과 함께 break문도 작성하였다.

 else {

    break;
}

이후 코드에서 breakN이 true이면 NO를 아니면 YES를 출력하는 식으로 작성하였다.  

해당 코드에는 2가지 문제점이 존재하는데   
1. 문자 비교가 끝나고 stack이 비어있지 않은 경우를 고려하지 않는다.  
   VPS는 () 쌍이 모두 이러어져있는 문자열인데 "()()(((" 이런식으로 쌍이 맞지 않지만 정상적으로 반복을 끝내는 경우가 존재한다.  
   이 경우 stack에는 (((이 남아있지만 위의 코드에서는 고려하지 않았기 때문에 YES가 나올 것이다.  
2. 1번 경우보다 더 심각한 문제는 stack을 초기화 시켜주지 않는데 있다.  
   n만큼의 문자열을 입력받고 각각 비교해서 출력하는데 처음 반복을 끝냈을때 stack이 비어있지 않다면 stack의 값이 누적되어  
   설계한 것과는 다르게 출력될 가능성이 높아진다.  

# 💻 코드 리뷰

```diff
import java.util.*;

public class Main{
    public static void main(String args[]){
        Stack<Character> stack=new Stack<>();
        Scanner s=new Scanner(System.in);
        StringBuilder sb=new StringBuilder();
        String answer="YES";
        
        int n=s.nextInt();
        boolean breakN=false;
        for(int i=0;i<n;i++){
            breakN=false;
            String str=s.next();
            for(int j=0;j<str.length();j++){
                char c=str.charAt(j);
                if(stack.isEmpty()){
                    if(c=='(')stack.push(c);
                    else{
                        breakN=true;
                        break;
                    }
                }
                else{
                    if(stack.peek()=='('&&c==')')stack.pop();
                    else if(stack.peek()=='('&&c=='(')stack.push(c);
                    else {
                        breakN=true;
                        break;
                    }
                }
            }
            if(breakN||!stack.isEmpty())answer="NO";
            else answer="YES";
            stack.clear();
            sb.append(answer).append("\n");
        }
        System.out.print(sb);
    }
}
```

해당 코드가 초기 코드의 문제점을 해결한 최종 코드이다.

```
if(breakN||!stack.isEmpty())answer="NO";
```
최종 코드에서는 반복문이 끝났을 때 stack이 비어있지 않다면 VPS아니기 때문에 위와 같이 비어있지 않다면 NO를 출력 가능하도록 수정하였다.

```
stack.clear();
```

또 n만큼 반복할때 stack이 비어있지 않다면 누적되어 의도한 바와 다르게 출력될 수 있는 가능성을 없애기 위해서  
매 반복이 끝난면 stack을 초기화 시켜주도록 수정하여 문제를 해결하였다.
